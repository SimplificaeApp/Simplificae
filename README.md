# Financeiro APP

## Descrição Curta
Aplicativo completo para gestão financeira empresarial e pessoal, integrado como um produto do ecossistema Simplificae.

## Stack Principal
- Next.js 15+ (App Router)
- React 19
- Tailwind CSS v4
- Supabase (Auth, DB)
- Framer Motion
- Lucide React

## Como instalar
1. Clone o repositório
2. Execute `npm install` na raiz do projeto
3. Configure as variáveis de ambiente baseadas no arquivo `.env.example`

## Como executar
1. Rode `npm run dev` para iniciar o servidor de desenvolvimento
2. Acesse http://localhost:3000

## Como rodar testes
*(Testes em implementação)*
- Unitários/Integração: `npm run test` (futuro)
- E2E: `npm run test:e2e` (futuro)

## Variáveis Obrigatórias
Consulte o arquivo `.env.example` para as variáveis de ambiente necessárias.
