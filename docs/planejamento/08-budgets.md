# Orçamentos (Budgets)

O módulo de orçamentos garante que o usuário não ultrapasse os limites estabelecidos para certas categorias.

## Funcionamento
1. O usuário define (ou a IA sugere) um valor máximo para uma categoria no mês (ex: R$ 800 para Restaurantes/Delivery).
2. Conforme as despesas vão sendo categorizadas, o sistema preenche a barra de progresso.
3. Se atingir 80%, o Agente Autônomo dispara um alerta proativo.

## Flexibilidade
- Um mês que economizou R$ 100 de Delivery pode (se o usuário desejar) rolar esse saldo extra de orçamento para o mês seguinte.
- Suporte a orçamentos globais (MEI: "Total de Custos Fixos Máximo R$ 5000/mês").
