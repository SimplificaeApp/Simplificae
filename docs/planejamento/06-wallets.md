# Carteiras (Wallets)

Carteiras são divisões lógicas do dinheiro do usuário, úteis para quem adota métodos como o "Sistema de Envelopes" (ex: "Envelope para Lazer", "Envelope para Investimento"). Diferem de Contas pois não representam instituições financeiras reais.

## Uso Prático
- Pode ser atrelada a uma Conta ou funcionar como um balde global.
- Útil para o usuário separar mentalmente o montante que está "livre" do montante "comprometido".
- (Este módulo será habilitado nas versões futuras V2, inicialmente as Metas suprirão essa divisão).
