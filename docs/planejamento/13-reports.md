# Relatórios (Reports)

Onde os dados brutos viram análises poderosas para Pessoas Físicas e DREs para as MEIs.

## 1. DRE Simplificado (Para MEIs e Pequenas Empresas)
Exibição do fluxo puro:
- (+) Receitas Operacionais
- (-) Impostos/Taxas de Pagamento
- (-) Custos Variáveis
- (-) Despesas Fixas
- (=) Lucro Líquido.

## 2. Visão Patrimonial (Net Worth)
Gráfico histórico somando Saldo de Contas Bancárias + Investimentos - Faturas Abertas - Dívidas = Patrimônio Líquido atual.

## 3. Relatório de Evolução de Categoria
Gráfico de barras mostrando quanto gastou com "Lazer" ao longo dos últimos 12 meses, para identificar inflação de estilo de vida.
