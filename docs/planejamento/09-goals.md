# Metas (Goals)

Este módulo substitui a necessidade complexa de usar carteiras para poupança. Serve para materializar sonhos e objetivos financeiros.

## Parâmetros da Meta
- **Nome** (Ex: "Viagem Europa", "Reserva de Emergência").
- **Valor Alvo** (Ex: R$ 15.000).
- **Data Alvo** (Quando deseja atingir).
- **Conta Vinculada** (Opcional, diz onde o dinheiro guardado fisicamente está rendendo).

## Dinâmica de Crescimento
1. O usuário cria uma transação de "Investimento em Meta".
2. O Dashboard exibe um Card de Conquista gamificado conforme a barra de progresso avança (25%, 50%, 100%).
3. O Agente de IA pode sugerir: "Você teve uma sobra de caixa de R$ 300, deseja alocar 50% disso na Meta 'Viagem Europa'?"
