# Assinaturas (Subscriptions)

Para resolver a dor da "morte por mil cortes" (pequenas assinaturas que somam muito).

## Monitoramento Automático
Em vez de depender de cadastro manual, o FinanceOS utiliza o **Agente Autônomo** para ler padrões de transação:
- Se "Netflix", "Spotify", "AWS" ou "ChatGPT" ocorre todo mês com variação de data pequena e nome constante, o sistema o isola em uma tela de "Assinaturas".
- Exibe o custo total anual acumulado dessas pequenas taxas.

## Gestão de Aumentos
- A IA avisa imediatamente se o plano de um serviço saltou de preço sem que o usuário notasse no extrato massivo.
